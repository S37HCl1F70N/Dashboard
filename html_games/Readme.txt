# HTML Games

Welcome to the **html_games** folder! This directory contains a collection of simple games built using HTML, CSS, and JavaScript. Each game is self-contained and can be played directly in your web browser.

## What You'll Find Here

- **Mini-Games**: Each file or subfolder is a standalone game. These are perfect for learning, experimentation, or just having fun!
- **Open Source**: Feel free to modify, enhance, or use these games in your own projects.

## How to Play

1. **Clone or Download** this repository to your local machine.
2. **Navigate** to the `html_games` folder.
3. **Open any `.html` file** in your web browser to start playing!

## Adding Your Own Game

Want to contribute? Follow these steps:
1. Create a new folder or file for your game (e.g., `myCoolGame.html` or `myCoolGame/`).
2. Use only HTML, CSS, and JavaScript (no server code required).
3. Test your game in a browser.
4. Submit a pull request!
